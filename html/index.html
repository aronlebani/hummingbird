Help

Keyboard shortcuts

f = Focus search bar

Esc = Focus page

h = Open help

Tab = Navigate through bookmarks

Enter = Open bookmark

Cmd + Enter = Open bookmark in new tab

Ignore

Bookmarks can be excluded from Hummingbird by placing them in a folder called 'Ignore' (case insensitive).

Managing bookmarks

Bookmarks can be created, edited, and deleted in the usual way using Chrome's bookmark manager (or a third party alternative).
